# Improve layout structure

NOTE: Most of the contents have already been implemented, do not try to recreate existing contents but rather improve the current functionality of the application.

Divide the requirements here into BE and FE changes.

The layout of the application should be as follows:

2. Esitykset / Votings tab
   * Query specficic votings by the title or description. Usually voting points consists of multiple votings which are held either at same date or different dates.
   * Votings should at least show:
     * The description what is being voted on and link to the section
     * How each parliament member voted
     * How parties have voted (and if _ryhmäkuri_ was not followed)
     * How government / opposition have voted
     * What other votings are connected to this (1. äänestys, 2nd. äänestys)
     * Each mietintö should display:
       * Who proposed
       * What was the proposal
3. Members of Parliament tab (Composition)
  * Show the people currently sitting in the parliament
  * Show in a list, sortable by name, party etc.
  * Available to be searchable by date (i.e. in time-domain), e.g. show composition of parliament in 1950
  * Each member should be possible to be opened for additional details
4. Member of parliament subpage (or popuop)
  * Show basic information
  * Show held governmental or trust positions
  * Show times when have been in parliament, when dropped out or joined
    * Show also reason why joined or left (or who substituted whom)
5. Analysis tab
  * TO BE DONE LATER
6. Admin tab
  * Visible ONLY in DEV mode
  * Shows the data sources (tables) in order to update the main data
  * Tables have fetch and parse status
    * Fetch updates information from datasource
    * Parse converts the raw data into more structured format
  * Clicking generate should generate the main database from scratch
7. Valiokunnat tab
  * Shows information about different valiokunnat (e.g. Suuri valiokunta, Maa-metsätalous etc.)
  * Sessions, who has been asked to talk about a topic (especially interesting the _company_ of the person to point out lobbying).
